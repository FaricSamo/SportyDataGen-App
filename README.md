# SportyDataGen-App

# Installation
To install SportyDataGen application (ideally into a virtualenv) just download the code (https://github.com/FaricSamo/SportyDataGen-App.git) and run the setup.py script with the install parameter. It will install application into the virtualenvâ€™s site-packages folder and also download and install all dependencies:

```console
$ python setup.py install
```

# Quickstart
Run the app with:

```console
$ flask run
```
